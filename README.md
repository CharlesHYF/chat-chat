# chat-chat
一个基于Java语言开发的简单聊天室

分为两个模块：客户端[chat-chat-client]、服务端[chat-chat-server]

通过Socket进行通信



## 技术栈

客户端：原生JDK

服务端：SpringBoot、MySQL、MyBatis-plus、Druid、Lombok、Jbcrypt、Log4J
